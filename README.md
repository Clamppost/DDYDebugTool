#DDYDebugTool

* 显示调试工具视图 
[DDYDebugTool show:YES];

* 在屏幕上要显示的内容
[DDYDebugTool log:@"点击按钮"];

*捕获系统打印转存到文件
[DDYDebugTool handleSystemLog];
